2. **Select your WhatsApp Business API app**
3. **Navigate to WhatsApp → Configuration**
4. **Set Webhook URL**: 
   ```
   https://your-stackblitz-url.stackblitz.io/api/webhook
   ```
5. **Set Verify Token**:
